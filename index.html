My Work

AllPack: Travel App

The goal of this project was to build a mobile app that helps users to plan and book their trip in a post-pandemic world. AllPack provides support to young worker travellers to solve lack of time to plan and organize all their research for their trips. The App joins all information in a very convenient way to the User and facilitates all planning.

Restyle: Mobile App Group Project

Restyle is a sustainable fashion and style app that provides its users with local sustainable fashion stores online, and features a community board for styling inspiration, reference and thrifting.

Canadian Government Website Redesign

This project aims to redesign Immigration, Refugee and Citizenship Canada Website. Through User Reserach, it was identified that there is a lot information displayed in the main page, overwhelming User. The website was redesigned to to be more user-friendly, organized, and intitutive.

The 519 Website Redesign

The519.org provides numerous useful programs for the LGBTQ+ community.The objective of this project was redesigned The 519 homepage so that the main purpose and goal of the organization stands out. Also, reorganized the primary navigation so that users can easily access the programs offered by the organization.
